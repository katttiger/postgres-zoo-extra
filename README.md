**Introduction:**
A Java Spring Boot webbapplication displaying the properties of different animals.

- Backend: PostgreSQL (hosted via Neon) as database, environment variables for configuration of database.Hosted on Tomcat.
- Frontend: Spring Boot Web and Thymeleaf (HTML).

Developed with IntelliJ.
 
**Developers:**
Ana, Axel, Cecilia and Ellinor.

**To start project:** 
Run by starting main in class PostrgresZooApplication.
 
**Environment variables:** 
Found and set in upper left corner (more actions (three dots) -> edit -> "Environment Variables" (file icon to the left in the input field)).
